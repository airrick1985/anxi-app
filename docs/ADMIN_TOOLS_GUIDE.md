# 管理員工具中心使用指南

## 📋 概述

管理員工具中心是為系統管理員和超級管理員設計的工具集，用於進行系統資料檢查和維護。

## 🔐 權限要求

- **系統管理員** (系統管理員 Role)
- **超級管理員** (超級管理員 Role)

⚠️ 其他角色無法訪問此工具中心。

## 🚀 快速開始

### 1. 訪問管理員工具中心

**方式一：從主頁進入**
1. 登入系統，進入首頁（/home）
2. 尋找「管理員工具」按鈕（僅系統管理員和超級管理員可見）
3. 點擊進入工具中心

**方式二：直接訪問 URL**
```
http://localhost:5173/#/admin-tools
```

### 2. 工具中心介面

管理員工具中心提供以下功能：

#### 📊 工具卡片
- **報告上傳檢查** ✅ 已實現
- **數據恢復工具** 🚧 即將推出
- **批量操作工具** 🚧 即將推出

## 🔍 報告上傳檢查工具詳細使用

### 功能說明

此工具用於檢查和修復以下數據不一致情況：

**檢查條件：**
找出符合下列所有條件的預約：
- ✓ `households` 文檔有 `inspectionReportUrl`（已上傳報告）
- ✓ `appointments` 的 `projectId` 和 `unitId` 相符
- ✓ `bookingType` = 「初驗」或「複驗」
- ✓ `reportUploaded` = `false`（未標記為已上傳）
- ✓ `status` = 「已完成」（預約已完成）

### 使用步驟

#### 步驟 1：輸入建案 ID
```
建案 ID: fuyu1129
```

#### 步驟 2：開始檢查
點擊「🔍 開始檢查」按鈕

#### 步驟 3：查看檢查結果

結果卡片顯示：
- **總戶別數**：該建案的所有戶別
- **有報告的戶別**：已上傳報告的戶別數
- **不匹配項**：發現的數據不一致項目
- **完整率**：報告完整率百分比

#### 步驟 4：查看詳細清單

表格顯示每個不匹配項的詳細信息：
| 欄位 | 說明 |
|-----|------|
| 戶別 | 預約所屬的戶別代碼 |
| 預約代碼 | 預約的唯一編碼 |
| 預約類型 | 初驗 或 複驗 |
| 預約人 | 進行預約的人名 |
| 報告數 | 上傳的報告數量 |
| 操作 | 複製 Appointment ID |

#### 步驟 5：修復不匹配項

**確認修復前：**
1. 仔細檢查清單中的預約資訊
2. 確保這些預約確實已上傳報告
3. 確認無誤後再進行修復

**執行修復：**
1. 點擊「🔧 確認修復全部」按鈕
2. 確認修復對話框
3. 系統自動更新所有不匹配項的 `reportUploaded` 為 `true`

**修復結果：**
成功修復後，頁面會自動重新檢查，確保資料已更新。

### 實際例子

#### 例子：檢查建案 fuyu1129

```
1. 輸入建案 ID: fuyu1129
2. 點擊「開始檢查」
3. 檢查結果：
   - 總戶別數: 120
   - 有報告戶別: 45
   - 不匹配項: 3
   - 完整率: 93%

4. 檢查詳細清單：
   ├─ 戶別: B7-09, 預約代碼: BC20260413001, 類型: 初驗
   ├─ 戶別: A01-04, 預約代碼: BC20260413002, 類型: 複驗
   └─ 戶別: C3-12, 預約代碼: BC20260413003, 類型: 初驗

5. 確認這 3 筆預約都有報告已上傳
6. 點擊「確認修復全部 (3)」
7. 修復完成，自動重新檢查確認
```

```

├── 工具選擇卡片

    ├── 詳細結果表格
    ├── 修復操作區

```

### 後端 Cloud Functions

1. **checkReportUploadMismatch**
   - 檢查不匹配項
   - 參數：projectId
   - 返回：檢查結果詳情

2. **fixReportUploadMismatch**
   - 修復不匹配項
   - 參數：appointmentIds 陣列
   - 返回：修復結果統計

## 🔄 後續擴充方案

### 如何添加新的管理員工具

1. **創建工具組件**
   ```
   src/components/AdminTools/NewTool.vue
   ```

2. **在 AdminToolsCenter.vue 中註冊**
   ```vue
   <!-- 添加工具卡片 -->
   <v-col cols="12" md="6" lg="4">
     <v-card @click="selectTool('newTool')">
       <!-- 卡片內容 -->
     </v-card>
   </v-col>

   <!-- 添加工具渲染 -->
   <NewTool v-if="selectedTool === 'newTool'" />

   <!-- 更新 getToolTitle 函數 -->
   newTool: '新工具名稱',
   ```

3. **導入新組件**
   ```javascript
   import NewTool from '@/components/AdminTools/NewTool.vue';
   ```

## 🐛 常見問題

### Q1：我看不到「管理員工具」按鈕
**A：** 檢查您的角色是否包含「系統管理員」或「超級管理員」。普通用戶無法訪問此功能。

### Q2：檢查後找到不匹配項，是否一定要修復？
**A：** 不一定。您可以先檢查清單中的預約詳情，確認這些預約確實已上傳報告後再決定是否修復。

### Q3：修復操作是否可以撤銷？
**A：** 修復操作會直接更新數據庫，無法撤銷。執行前請確保檢查無誤。

### Q4：如何找到特定預約的詳細信息？
**A：** 在詳細清單表格中，您可以：
- 複製 Appointment ID
- 在 Firestore 中查詢
- 或在系統中搜尋該預約
